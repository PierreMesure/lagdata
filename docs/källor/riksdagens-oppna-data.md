# Riksdagens öppna data

Riksdagen publicerar sedan 2010 en stor mängd rättsinformation som öppna data. Institutionen har varit ledande även om den har fortfarande svårt att publicera andra viktiga uppgifter som riksdagsledamöternas ekonomiska intressen (som behöver fortfarande begäras).

Riksdagen publicerar information om lagarna och den parlamentariska delen av lagstiftningsprocessen. Men de har även tagit på sig ansvaret att publicera förarbeten som Regeringskansliet vägrar att tillgängliggöra som öppna data: SOU, Ds, Kommittédirektiv och även den svenska författningssamlingen.

Regeringskansliet publicerar tyvärr enbart förarbetena som PDF-dokument och Riksdagsförvaltningen konverterar dem till HTML men kvalitéten räcker inte alltid till alla användningsfall.

## API

Riksdagens API är inte mycket dokumenterade men det finns ett [enkelt sökgränssnitt](https://data.riksdagen.se/dokumentlista/) för att skapa en sökfråga och när man har hittat rätt kan man kopiera webbadressen och återanvända den. [Arkivfiler](https://www.riksdagen.se/sv/dokument-och-lagar/riksdagens-oppna-data/dokument/#dataset-10) finns också för att ladda ner stora mängder historisk data.
