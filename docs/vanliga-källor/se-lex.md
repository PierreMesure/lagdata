# SE-Lex

[SE-Lex](https://github.com/se-lex) är ett projekt från Martin Rimskog som samlar lagen som ett git-repository.

Projektets syfte är att tillgängliggöra svenska författningssamlingen i Markdown-format och versionerat. Git har många fördelar för det, det gör det enkelt att bläddra i lagarna vid en viss tidpunkt, se exakt vilka ord förändras med varje ny författning.

SE-Lex stödjer också export av SFS till HTML-version med ELI och Markdown med temporala taggar, som kan vara en lämpligt material om man vill göra dataanalyser på SFS.
