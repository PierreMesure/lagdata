# Propositioner

## Officiel källa

Regeringskansliet ([regeringen.se > Dokument & publikationer > Proposition](https://www.regeringen.se/rattsliga-dokument/proposition/))

### Erbjuder de ett API?

❌ Nej

## Alternativa källor

### Riksdagens öppna data

Se [den relevanta sidan](../vanliga-källor/riksdagens-oppna-data).

Riksdagsförvaltningen samlar alla propositioner anda sedan 1867.

Observera att det kan dröja flera dagar eller veckor innan vissa dokument hittas i Riksdagens öppna data.

### g0v.se

Se [den relevanta sidan](../vanliga-källor/g0vse).

g0v.se erbjuder en strukturerad lista av alla propositioner som går att ladda ner på regeringen.se med max 24 timmar fördröjning. De äldsta som är tillgängliga där är från 1996.

### Lagen.nu

Se [den relevanta sidan](../vanliga-källor/g0vse).

Lagen.nu tillgängliggör propositioner anda sedan 1867.
