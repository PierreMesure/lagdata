# Lagdata

> Lagrummet.se for developers

Sweden has no public legal information system (*rättsinformationssystem*). This means that there is no single place where someone can find a legal information in an open, structured and machine-readable way.

Lagdata is a collaborative documentation to explain how to access and reuse legal information.

## Run locally

Using [uv](https://docs.astral.sh/uv/):

```bash
uv run zensical serve
```
